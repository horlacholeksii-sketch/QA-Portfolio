# Bug Report: UI Elements Overlay in "Super Offer" Window

**Project:** Bingo: Love in Montana  
**Severity:** Low / UI  
**Environment:** Samsung M14 (Android 15), Resolution: 1080 x 2408   

---

## Summary
UI decorative elements (background graphics) overlap with the main "Super Offer" window frame, causing visual inconsistency.

## Steps to Reproduce
1. Launch the app.
2. Tap on the "Super Offer" icon on the left side of the screen.

## Expected Result
UI elements are correctly layered; decorative graphics should stay behind the main window frame.

## Actual Result
Decorative elements overlap the window frame, distracting the user and degrading the visual quality of the offer.
